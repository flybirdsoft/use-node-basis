# use-node-basis
作者：邬畏畏

注释：此项目是 nodejs 项目

（1）先把项目clone到本地

（2）进入目录,运行命令如下：

    npm install
    
    npm install 会安装node-basis依赖包 ,node-basis是本人开发的可以替代Express包
    
    
 (3)运行命令 
    npm start
